СПОСОБЫ ЗАПУСКА (выбери удобный)
─────────────────────────────────────

1) «Polygraph (без окна).vbs»  ← ОБЫЧНОЕ ИСПОЛЬЗОВАНИЕ
   - Сервер в фоне, окна нет. Браузер откроется сам.
   - Остановить: двойной клик «Остановить Polygraph.bat».

2) «Запустить Polygraph.bat»
   - С чёрным окном (видно сервер). Остановить — закрыть окно.

3) «Polygraph (разработка).bat»  ← ДЛЯ ИЗМЕНЕНИЯ КОДА
   - Сервер САМ подхватывает правки в .py-файлах.
   - Поменял код -> сохранил -> просто обнови страницу (F5).
   - Перезапускать НЕ нужно! Удобно при доработке.
   - Остановить — закрыть окно.

ОСТАНОВКА СЕРВЕРА
   - «Остановить Polygraph.bat» — убивает сервер без Диспетчера задач.

⚠️ Перед первым запуском:
   - Включи VPN (для Groq).
   - Если не установлен Flask:  pip install flask


КРАСИВЫЙ ЯРЛЫК С ИКОНКОЙ НА РАБОЧЕМ СТОЛЕ
──────────────────────────────────────────
1. Правой кнопкой по файлу запуска (.bat или .vbs)
   → «Отправить» → «Рабочий стол (создать ярлык)».
2. Правой кнопкой по ярлыку → «Свойства».
3. «Сменить значок...» → «Обзор...» →
   выбери:  C:\Полиграф\polygraph\polygraph.ico
4. «ОК» → «Применить» → «ОК».
5. (по желанию) Переименуй ярлык в «Polygraph».

Готово! Запуск двойным кликом по иконке. 💎


ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ
────────────────────────
• «python не является командой» — установи Python с python.org
  (галочка "Add Python to PATH" при установке).
• «No module named flask» —  pip install flask
• Браузер не открылся — открой вручную:  http://127.0.0.1:5000
• Forbidden / агент не отвечает — проверь VPN (для Groq).
• «port in use» — сервер уже запущен в другом окне, используй его.
