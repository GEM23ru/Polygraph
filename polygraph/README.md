# 🔷 Polygraph

Мульти-провайдерный AI-агент с роутингом и инструментами.
Сам выбирает подходящую модель под задачу и умеет искать, считать,
работать с файлами и выполнять код.

> 🔄 **Передача дел / онбординг нового агента:** см. `HANDOFF.md` —
> там полная история работы, грабли и план дальнейших действий.

## Установка

```bash
pip install -r requirements.txt
cp .env.example .env      # впиши свои ключи (нужен хотя бы один)
```

Бесплатные провайдеры, которых хватит на старт:
- **OpenRouter** — много free-моделей → https://openrouter.ai/keys
- **Google AI Studio (Gemini)** — лучший function calling → https://aistudio.google.com/apikey
- (опц.) **Groq** — очень быстрый → https://console.groq.com/keys

## Запуск

```bash
python web.py             # 🌐 веб-чат в браузере (http://127.0.0.1:5000) — удобнее всего
python chat.py            # 💬 чат в консоли
python chat.py --debug    # с логами вызовов инструментов

python healthcheck.py     # 🩺 проверить, какие модели живы с твоими ключами
python healthcheck.py --free   # только бесплатные

python run.py             # демо всех возможностей (роутинг, ансамбль и т.д.)
```

## Что умеет агент

| Инструмент | Действие |
|---|---|
| `web_search` | Поиск в интернете (DuckDuckGo бесплатно, Tavily если есть ключ) |
| `calculator` | Математические вычисления |
| `run_python` | Выполнить Python-код и вернуть результат |
| `fetch_url` | Открыть веб-страницу по ссылке и прочитать её текст |
| `read_file` / `write_file` / `list_files` | Работа с файлами (в папке `agent_files/`) |
| `datetime_now` | Текущие дата и время |

> ⚠️ `fetch_url` читает обычные страницы (статьи, документация, блоги).
> Сайты с защитой от ботов и JS-рендерингом (Авито, маркетплейсы,
> соцсети) часто отдают капчу/пустую страницу — это ограничение
> простого парсера, а не баг.

## Возможности ядра (core.py)

- **Роутинг** — задача автоматически уходит на подходящую модель
- **Fallback** — если модель упала, пробуется следующая
- **Кэш** — повторные запросы не тратят лимиты (5 мин)
- **Параллельные вызовы**, **ансамбль** (синтез из нескольких моделей), **load-balancing**

## Команды в чате
`/help` `/tools` `/models` `/clear` `/stats` `/exit`

## Про бесплатные лимиты
- Google Gemini free: **10 запросов/мин**
- OpenRouter free: **~20/мин, ~50/день**

Ошибки `429` = лимит исчерпан (подожди), `503` = провайдер занят. Это не баг.
